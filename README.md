# linkedart

Library of functions for parsing Linked Art (https://linked.art/) data.

## Project setup

```
npm install
```

### Compiles and minifies for production

```
npm run build
```

### Build the JSDoc

```
npm run build:jsdoc
```

### Run your tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```
